# 🎮 PlayBase

PlayBase é uma plataforma web de jogos desenvolvida com o objetivo de centralizar e organizar conteúdos interativos em um único site. O projeto foi criado para prática de desenvolvimento full stack, integrando front-end, back-end e banco de dados.

---

## 🚀 Tecnologias utilizadas

- Python 🐍  
- Flask 🌐  
- HTML5 🎨  
- CSS3 🎨  
- JavaScript ⚙️  
- MySQL 🗄️  

---

## 📌 Funcionalidades

- Sistema de login e cadastro de usuários  
- Catálogo de jogos disponível na plataforma  
- Interface responsiva e intuitiva  
- Integração completa com banco de dados MySQL  
- Estrutura dinâmica com Flask (rotas e templates)  
- Interações e melhorias de UX com JavaScript  

---

## 🧠 Objetivo do projeto

O PlayBase foi desenvolvido com o objetivo de praticar e consolidar conhecimentos em:

- Desenvolvimento web full stack  
- Criação de aplicações com Flask  
- Integração com banco de dados relacional (MySQL)  
- Desenvolvimento de interfaces modernas com HTML, CSS e JavaScript  
- Organização e estruturação de projetos reais
